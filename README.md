# Shajar Ali - Professional Portfolio

A high-performance, creative portfolio website designed to showcase expertise in frontend and blockchain development. This project focuses on delivering a premium user experience through fluid animations, dynamic scrolling effects, and a modern aesthetic.

## Overview

This portfolio serves as a digital showcase for Shajar Ali, a Lahore-based developer specializing in bridging the gap between Web2 and Web3. The application is built with a focus on performance, responsiveness, and interactive storytelling.

## Core Technologies

The project leverages a modern technology stack to ensure scalability and a refined user interface:

- **Framework**: Next.js (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Language**: JavaScript (ES6+)

## Key Features

- **Dynamic Parallax Scrolling**: Implements advanced scroll-linked animations using Framer Motion for an immersive experience.
- **Responsive Design**: Fully optimized for all screen sizes, from mobile devices to large desktop monitors.
- **Interactive Work Section**: A dedicated component to showcase projects with smooth transitions and hover effects.
- **Real-time Local Information**: Displays live local time and location data for a personalized touch.
- **Fluid UI/UX**: Custom preloader and background transitions that adapt as the user navigates through different sections.

## Getting Started

### Prerequisites

Ensure you have the following installed on your local machine:

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone [repository-url]
   ```

2. Navigate to the project directory:

   ```bash
   cd portfolio
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

### Development

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:3000`.

### Building for Production

To create an optimized production build:

```bash
npm run build
```

To start the production server:

```bash
npm run start
```

## Project Structure

- `src/app`: Contains the main application routes and global styles.
- `src/components`: Reusable UI components such as Navbar, WorkSection, and ContactSection.
- `public`: Static assets including images and documents.

## Deployment

This project is optimized for deployment on the Vercel platform. For detailed deployment instructions, refer to the Next.js documentation.
